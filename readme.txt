== Description ==

Twilio SMS Notifications lets you send automatic SMS updates to customers based on WooCommerce order status changes.

**Features:**
- Admin settings for Twilio SID, Auth Token, and sender number
- Message templates for each WooCommerce order status
- Support for placeholders: {customer_name}, {order_id}, {order_total}, {site_name}
- Works globally with verified Twilio numbers

== Installation ==

1. Upload the plugin to the `/wp-content/plugins/` directory or install via Plugins > Add New
2. Activate the plugin
3. Go to **Twilio SMS** menu in the WordPress dashboard
4. Enter your Twilio credentials and customize message templates

== Frequently Asked Questions ==

= Can I use this on a Twilio trial account? =
Yes, but you can only send messages to verified numbers.

= What are the supported placeholders? =
You can use: {customer_name}, {order_id}, {order_total}, {site_name}
